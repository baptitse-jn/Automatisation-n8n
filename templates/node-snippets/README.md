## Comment utiliser

1. Ouvrir le fichier JSON du snippet
2. Copier la configuration du node
3. Dans n8n, utiliser "Import from JSON" ou coller dans l'éditeur

## Comment contribuer

Lors de l'ajout d'un snippet :
1. Utiliser un nom descriptif : `category-action.json`
2. Inclure des commentaires dans le JSON si possible
3. Documenter les credentials nécessaires
4. Mettre à jour ce README
