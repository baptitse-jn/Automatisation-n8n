# Ressources Humaines

Workflows d'automatisation pour la gestion RH.

## Catégories

| Dossier | Description | Cas d'usage |
|---------|-------------|-------------|
| `recruitment/` | Recrutement | Tri CV, sourcing, suivi candidats |
| `onboarding/` | Onboarding | Accueil nouveaux employés, checklist |
| `employee-management/` | Gestion employés | Congés, évaluations, formation |

## Intégrations courantes

- **ATS** : Lever, Greenhouse, Welcome to the Jungle
- **SIRH** : BambooHR, Personio, PayFit
- **Communication** : Slack, Teams
- **Signature** : DocuSign, Yousign

## Comment ajouter un workflow

1. Placer le fichier `.json` dans le sous-dossier approprié
2. Créer un `README.md` dans le même dossier avec la documentation
3. Mettre à jour ce fichier avec les infos du workflow
