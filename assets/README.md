# Assets

Ressources visuelles pour la documentation et les workflows.

## Structure

```
assets/
├── diagrams/       # Diagrammes d'architecture
├── screenshots/    # Captures d'écran des workflows
└── icons/          # Icônes personnalisées
```

## Conventions de nommage

- Utiliser des noms descriptifs en kebab-case
- Préfixer par le domaine : `marketing-email-flow.png`
- Inclure la date si pertinent : `dashboard-2024-01.png`

## Formats recommandés

| Type | Format | Usage |
|------|--------|-------|
| Diagrammes | `.svg`, `.png` | Architecture, flows |
| Screenshots | `.png` | Documentation |
| Icônes | `.svg` | UI, branding |

## Outils recommandés

- **Diagrammes** : Excalidraw, Mermaid, Draw.io
- **Screenshots** : CleanShot X, ShareX
- **Édition** : Figma, Canva
